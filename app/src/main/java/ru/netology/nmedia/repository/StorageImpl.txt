            ХРАНЕНИЕ ДАННЫХ

    1. SHARED PREFERENCES

    Сериализация - это преобразование данных из одного формата в другой. На практике под
сериализацией понимается преобразование объекта из одного формата в набор байтов или в строку.
Другими словами, сериализация - это адаптация данных произвольного формата в формат строк либо
в байтовый поток, и наоборот.
    Application - это базовый класс для управления глобальным состоянием приложения (он также
является контекстом, но про него говорят, что это контекст приложения). Добраться до этого
контекста из любого другого контекста можно с помощью вызова функции getApplicationContext().
